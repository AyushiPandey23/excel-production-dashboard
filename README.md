# Production Data Analysis Dashboard (Excel)

## Problem Statement

This project focuses on analyzing production data to monitor overall performance and support data-driven decision-making.  
The dashboard helps in tracking production output, cost efficiency, and managerial performance.  

It also enables comparison across product categories and identification of trends over time, helping businesses improve efficiency and productivity.

---

## Steps Followed

- Step 1 : Imported and cleaned the dataset in Microsoft Excel  
- Step 2 : Performed data transformation for better analysis  
- Step 3 : Created Pivot Tables for summarizing production data  
- Step 4 : Designed an interactive dashboard layout  

- Step 5 : Created visuals:
  - Production performance analysis  
  - Cost efficiency tracking  
  - Category-wise performance comparison  
  - Manager-wise productivity analysis  

- Step 6 : Added filters/slicers for interactive data exploration  

- Step 7 : Organized dashboard for clear and user-friendly insights
